# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |

## Reporting a Vulnerability

**Please use [GitHub Private Vulnerability Reporting](https://github.com/axelhamil/aggregate-kit/security/advisories/new)** to report security issues.

Do **not** open a public issue for security vulnerabilities.

### What to expect

- **Response time:** within 1 week (best-effort, solo maintainer)
- **Fix timeline:** depends on severity; patches for confirmed issues are prioritized

### Scope

This package ships with **zero runtime dependencies**, keeping the attack surface minimal. Reports about transitive dev-dependency vulnerabilities are appreciated but lower priority.
